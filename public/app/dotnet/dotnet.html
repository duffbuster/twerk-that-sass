.NET - Visual Studio 2015

Gulp & Task Runner Explorer

1. Create package.json file

Right click Solution Explorer. Choose Add... New Item. Under Visual C# / Web - Choose "NPM Configuration File"

2. Add dev dependancies to package.json

"devDependencies": {
    "bower": "^1.3.12",
    "gulp": "^3.9.1",
    "gulp-clean-css": "^2.0.8",
    "gulp-concat": "^2.6.0",
    "gulp-livereload": "^3.8.1",
    "gulp-rename": "^1.2.2",
    "gulp-rev": "^7.0.0",
    "gulp-sass": "^2.3.1",
    "main-bower-files": "^2.13.1",
    "run-sequence": "^1.2.1",
    "del": "^2.2.0"
  },

3. Install Packages

VS does this for you on save!

4. Create gulpfile.js

Right click Solution Explorer. Choose Add... New Item. Under Visual C# / Web - Choose "Gulp Configuration File"

5. Associate gulp.task to an appropriate binding

Show the Task Runner Explorer by choosing View: Other Windows: Task Runner Explorer

The tasks you've declared should show up on the left pane, if not hit refresh.
