Goals

Local & Dev Environment

  • CSS should compile either on save or page refresh
  • CSS should not be minified
  • Files should not be concatinated

Production

  • CSS should compile once on build and be cached.
  • CSS should be minified
  • Files should be concatinated
  • File names should include some token to prevent users' browsers from having cached styles.

Code Repository

It's best to keep built assets out of your code repo. Add all built CSS to .gitivnore or svn.ignore. This will keep merge conflicts to a minimum.
