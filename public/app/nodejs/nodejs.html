Node.js

Gulp

"The Streaming Build System"

Getting Started

1. Install Gulp

npm install --global gulp-cli

2. Create package.json file

npm init

3. Add dev dependancies to package.json

"devDependencies": {
    "bower": "^1.3.12",
    "gulp": "^3.9.1",
    "gulp-clean-css": "^2.0.8",
    "gulp-concat": "^2.6.0",
    "gulp-livereload": "^3.8.1",
    "gulp-rename": "^1.2.2",
    "gulp-rev": "^7.0.0",
    "gulp-sass": "^2.3.1",
    "main-bower-files": "^2.13.1",
    "run-sequence": "^1.2.1",
    "del": "^2.2.0"
  },
  • bower - Package manager for front end libraries
  • Gulp - "The Streaming Build System"
  • gulp-clean-css - minifies css
  • gulp-concat - concatinates many files into one
  • gulp-livereload - gulp implementation for the chrome plugin
  • gulp-rename - renames files
  • gulp-rev - adds hash to filenames
  • gulp-sass - compmiles Sass to CSS
  • main-bower-files - gets a list of main files in bower packages
  • run-sequence - forces tasks to run in order
  • del - deletes files from filesystem

4. Install Packages

npm install

5. Create gulpfile.js

  • gulp.src(globs) - Files to be acted on. Returns a stream that can be .pipe into series of plugins
  • gulp.dest(path) - Destination for results
  • gulp.task(name [deps] [fn]) - creates a task that can either be run in CLI, or run as a dependancy for another task
  • gulp.watch(glob, callback) - watch a set of files for changes, executes callback on save
