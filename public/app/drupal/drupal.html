Drupal 7

Zen Theme

1. Create Subtheme

drush zen [subtheme_name]

2. Install Packages

In your console, cd to sites/all/themes/[subtheme_name]

npm install

3. Run Gulp

For Dev:

gulp watch

For production, in your build process run 

gulp

Also in prod, go to Configuration: Performance, under Bandwidth Optimization - Check Aggregate and compress CSS

Resources:

  • Create a Zen 7.x-6.x sub-theme - drupal.org

Alternatives:

  • Gulp
  • Sassy - not recommended. PHPSass is no longer being maintained.
